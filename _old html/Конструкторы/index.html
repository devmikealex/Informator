Конструктор горки с шариками Track Park 212 деталей

Основная задача - построить горку так, чтобы шарик не застревал, а скатывался вниз по дорожкам и виражам. В игре могут участвовать несколько игроков одновременно, можно устраивать гонки соревнуясь чей шарик придёт к финишу первым. Просто запустите шарик, который начнёт катиться по желобам. Отличительный элемент конструктора - это катапульта, которая запускает шарик вверх по трубкам и он начинает своё движение по лабиринту.

Все элементы выполнены из качественного пластика, легко соединяются между собой и образуют прочную конструкцию. Особенностью набора являются подвижные элементы-препятствия, которые делают запуск шариков максимально увлекательными. Набор состоит из 152 пластиковых элементов (трубки, мельницы, воронки, извилистые дорожки и другие детали), инструкция по сборке, 60 стеклянных шариков. Способствует развитию мелкой моторики, креативного мышления, интеллектуальному развитию.

Конструктор Bauer Веселые горки 113 элементов

Конструктор Bauer серии Веселые горки подходит для малышей, мальчиков и девочек. С ним можно построить множество вариантов лабиринтов для скатывающегося вниз шарика, а затем наблюдать результаты своего труда, следя за тем, как шарик следует по заданной траектории. Пластиковый конструктор Bauer Веселые горки это игра, развивающая воображение и логику ребенка. Конструктор прекрасно подходит для детского творчества и семейного отдыха, способствуют развитию объёмного мышления ребенка, фантазии, координации движения глаз, мелкой моторики и памяти. В состав конструктора входит 113 деталей различных форм, цветов и размеров. Игрушки сделаны из высококачественного пластика с использованием пищевых красителей. Комплектация наборов конструктора входят желоба для шарика, шарики, опоры для горок, приемные желоба для шариков, составные детали горок с желобами и без, вращающиеся элементы с лопастями.

Конструктор Трубопровод детский 90 деталей

В комплекте 90 деталей : разноцветные трубочки и 4 колеса, с помощью которых можно строить разнообразные трубчатые лабиринты и автотехнику. Полёт фантазии безграничен. Такой набор на долго увлечет ребенка и принесет много пользы: поспособствует развитию мелкой моторики, логики, воображения. Малышам поможет познакомиться с основными цветами, счетом и размерами. Размер трубок от 4 до 7 см. Конструктор "Трубопровод" совершенно безопасен для малышей, детали крупные и ребенок не сможет их использовать не по назначению. С конструктором можно играть в воде. Трубы плотно скрепляются между собой и не пропускают воду. Конструктор немного туговат для сборки детям, может потребоваться помощь взрослых. Это сделано специально для игр в воде, что бы вода не подтекала из труб. В комплекте есть схема с разными вариантами сборки.
